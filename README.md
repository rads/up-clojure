# Clojure on Apex Up

This is an example project to deploy an HTTP server written in Clojure using [Apex Up](https://github.com/apex/up).

## Getting Started

Install Up:

```
$ curl -sf https://up.apex.sh/install | sh
```

Deploy the app:

```shell
$ up
```

Open it in the browser:

```shell
$ up url --open
```
